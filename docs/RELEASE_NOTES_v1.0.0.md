# FIC6 Torque Processor v1.0.0

Initial portable Windows release of the FIC6 Torque Processor.

## What's Included

- Portable Windows EXE: `FIC6_Torque_Processor.exe`
- Start guide for basic users: `START_HERE.txt`
- Source scripts organized under `src/`
- Build and setup scripts under `scripts/`
- Demo logs under `examples/`

## Highlights

- No Python install required for normal users
- GUI workflow for selecting FIC6 and Torque logs
- Processing buttons arranged in the intended workflow order
- Helper processing scripts bundled into the portable executable
- Dependencies bundled into the portable executable
- Outputs are written beside the selected log files

## Basic Usage

1. Download the release files.
2. Double-click `FIC6_Torque_Processor.exe`.
3. Select the FIC6 `.txt` log.
4. Select the Torque `.csv` log.
5. Run the buttons from top to bottom.

## Notes

Windows may show a SmartScreen warning because this EXE is not code-signed.
